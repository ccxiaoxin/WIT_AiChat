# 原始文档目录

此目录用于存放需要向量化的原始文档。

## 目录结构

```
doc/
├── pdfs/      # PDF 文档
├── txts/      # 文本文档
└── docs/      # Word 文档
```

## 使用说明

1. 将文档放入对应的子目录
2. 运行初始化脚本：`npm run init-vector`
3. 脚本会自动：
   - 读取文档内容
   - 进行文本切片
   - 向量化
   - 存储到向量数据库

## 支持的文档格式

- ✅ TXT：纯文本文件
- ✅ PDF：需要 pdf-parse 库
- ✅ DOCX：需要 mammoth 库
- ⚠️ 其他格式需要额外处理

## 示例文档

可以在这里放置：
- 课程大纲
- 专业介绍
- 就业指导手册
- 学校政策文件
- FAQ文档
等

## 注意事项

- 文档内容要清晰、结构化
- 避免大量重复内容
- 建议单个文档不超过100页
- 定期更新以保持知识库新鲜度
